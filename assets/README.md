# Assets Directory

This directory contains static assets for the website.

## Structure

- `images/` - Store all images, photos, and graphics here
  - Profile pictures
  - Project screenshots
  - Icons and logos
  
- `css/` - Custom CSS files (if needed)
  - Additional stylesheets
  - Custom animations
  
- `js/` - JavaScript files (if needed)
  - Custom scripts
  - Interactive components

## Usage

When adding images to your website, place them in `assets/images/` and reference them like:

```html
<img src="assets/images/profile.jpg" alt="Profile Picture">
```
